# Microservice Migration Guide (Full Version)

## Objective
Provide Copilot with full rules for analyzing legacy .NET monoliths and proposing a modern Spring Boot microservice architecture.

## Key Responsibilities
- Identify bounded contexts.
- Map domain entities to microservices.
- Extract responsibilities from Core.Services.
- Design REST endpoints replacing WCF/ASMX.
- Outline event-driven patterns.
- Define database ownership segregation.
