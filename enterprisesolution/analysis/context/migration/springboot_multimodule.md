# Spring Boot Multi-Module Migration Blueprint

## Overview
This blueprint defines how to structure the target Spring Boot multi-module architecture from a large .NET 4.6 codebase. It includes domain-driven boundaries, module responsibilities, data access modernization, and integration with Kafka, MQ, and CICS.

## Target Modules
- common-domain
- common-events
- customer-service
- order-service
- billing-service
- legacy-interop-service
